
> 이미지나 비디오 같은 대체 요소가 컨테이너에 어떻게 맞춰질지 결정하는 속성

## 속성 값

### `fill` (기본값)
- 컨테이너를 **완전히 채움**
- 원본 비율 무시하고 늘어나거나 찌그러짐
- 빈 공간 없음, 잘림 없음

### `contain`
- 원본 **비율 유지**
- 컨테이너 안에 **완전히 들어가도록** 축소/확대
- 빈 공간이 생길 수 있음
- 잘림 없음

### `cover`
- 원본 **비율 유지**
- 컨테이너를 **완전히 채움**
- 이미지/비디오 일부가 **잘릴 수 있음**
- 빈 공간 없음

### `none`
- **원본 크기** 그대로 유지
- 컨테이너 크기와 무관
- 잘리거나 빈 공간 생길 수 있음

### `scale-down`
- `none`과 `contain` 중 **더 작은 결과**를 선택
- 원본이 컨테이너보다 작으면 → `none` (원본 크기)
- 원본이 컨테이너보다 크면 → `contain` (축소)
