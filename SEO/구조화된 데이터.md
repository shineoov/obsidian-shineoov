
# 구조화된 데이터 ( Structured Data )

## 기본 개념
> 구조화된 데이터(**Structured Data**)는 검색 엔진이 웹페이지의 콘텐츠를 더 정확하게 이해할 수 있도록 돕는 표준화된 데이터 형식입니다.
> **스미카 마크업**이라고도 한다.

## Google의 설명
> "Google 검색은 페이지의 콘텐츠를 파악하기 위해 노력합니다. 페이지에 구조화된 데이터를 포함하면 Google에 페이지 의미에 관한 확실한 단서를 제공하여 내용을 파악하는 데 도움이 됩니다. 
> 구조화된 데이터는 페이지에 관한 정보를 제공하고 페이지 콘텐츠를 분류하기 위한 표준화된 형식으로 예를 들어 레시피 페이지의 경우 재료, 조리 시간, 온도, 칼로리 등이 여기에 해당합니다."

## 주요 특징
- **표준화된 형식**: Schema.org 기반의 일관된 데이터 구조
- **명확한 정보 전달**: 페이지 내용을 검색 엔진에 직접적으로 설명
- **리치 결과 생성**: 검색 결과에서 더 풍부한 정보 표시 가능

## 지원되는 형식
- **JSON-LD** (권장): JavaScript 객체 형태
- **Microdata**: HTML 태그에 직접 포함
- **RDFa**: HTML 속성으로 표현

## Product 스키마 필드 설명

### 기본 구조
- **@context**: Schema.org 표준 정의 (항상 `https://schema.org/` 사용)
- **@type**: 스키마 타입 (`Product` - 상품 페이지 선언)

### 상품 정보
- **name**: 상품명 (검색 결과에 표시)
- **description**: 상품 설명 (간단한 요약)
- **image**: 상품 이미지 URL (검색 결과 썸네일)

### 브랜드 정보
- **brand**: 브랜드 객체
  - `@type`: "Brand"
  - `name`: 브랜드명

### 판매 정보 (offers)
- **@type**: "Offer"
- **url**: 상품 구매 페이지 URL
- **priceCurrency**: 통화 단위 (예: KRW, USD)
- **price**: 가격 (숫자만, 쉼표 없이)
- **priceValidUntil**: 가격 유효 기간 (YYYY-MM-DD)
- **availability**: 재고 상태
  - `InStock`: 재고 있음
  - `OutOfStock`: 품절
  - `PreOrder`: 예약 판매
- **seller**: 판매자 정보
  - `@type`: "Organization"
  - `name`: 판매자명

### 평점 정보 (aggregateRating)
- **@type**: "AggregateRating"
- **ratingValue**: 평균 평점
- **reviewCount**: 총 리뷰 개수
- **bestRating**: 최고 점수
- **worstRating**: 최저 점수

### 적용 효과
- 검색 결과에 별점, 가격, 재고 상태 표시
- 상품 이미지가 검색 결과에 노출
- 클릭률(CTR) 향상


## 예시
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "인스타그램 팔로워, 좋아요 최적화 계정 활성화 관리해드립니다.",
  "description": "인스타그램 마케팅 활성화",
  "image": "https://img.adcheap.co.kr/images/5/9/59d9a5f9b5acb19eb9cb432153c87419.webp",
  "brand": {
    "@type": "Brand",
    "name": "애드칩"
  },
  "offers": {
    "@type": "Offer",
    "url": "https://adchip.co.kr/products/4",
    "priceCurrency": "KRW",
    "price": 5000,
    "availability": "https://schema.org/InStock",
    "seller": {
      "@type": "Organization",
      "name": "애드칩"
    }
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": 9,
    "reviewCount": 361,
    "bestRating": 10,
    "worstRating": 0
  },
  "review": [
    {
      "@type": "Review",
      "author": {
        "@type": "Person",
        "name": "평*****"
      },
      "datePublished": "2025-07-18",
      "reviewBody": "오늘도 빠르게 잘 이용했습니다! 감사합니다ㅎㅎ 만족!!",
      "reviewRating": {
        "@type": "Rating",
        "ratingValue": 10,
        "bestRating": 10,
        "worstRating": 0
      }
    }
  ]
}
</script>
```
